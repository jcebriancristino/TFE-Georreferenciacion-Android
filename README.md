Aplicación Android para georreferenciación de imágenes con fines tácticos
Este proyecto ha sido desarrollado como parte del Trabajo de Fin de Estudio del Grado en Ingeniería Informática de la Universidad Internacional de La Rioja (UNIR).

Autor
José Cebrián Cristino

Descripción
Aplicación Android orientada a la georreferenciación de imágenes con fines tácticos en entornos sin conexión a redes. La herramienta permite cargar una imagen, definir puntos de control conocidos, calcular una matriz de homografía, convertir coordenadas entre sistemas geográficos y UTM, representar simbología táctica e importar/exportar datos mediante archivos CSV.

Tecnologías utilizadas
Android Studio
Java
OpenCV
Proj4j
Arquitectura MVVM
